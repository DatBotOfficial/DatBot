# ⚙️ Confugration

### Server

* `/bot setup`
  * Use the setup for a server
  * Checks permissions and more.



### User

* `/user settings [choice] [setting]`
  * Set settings for your account
  * Argument \[choice]
    * Select a setting.
  * Argument \[setting]
    * Set the value for the \[choice] setting to.
